# كل مكون يصبح Entity حقيقية

- أسماء متعددة
- ترجمات
- لهجات
- صور
- حساسية
- Tags
- علاقات بالمطابخ
- مرادفات

# تجربة إضافة المكونات للمطعم

3. يختار المكون.
4. ينتقل للمكون التالي.
