# خارطة الطريق الأولية لمشروع بابلي

# المرحلة 1 — Core Foundation

الهدف:
إنشاء قلب بابلي الأساسي بأقل تعقيد ممكن.

## تشمل:

- حسابات المطاعم
- صفحات المطاعم
- منيو إلكتروني
- ترجمة متعددة اللغات
- QR للطاولات
- الطلبات داخل المطعم
- Kitchen Flow بسيط
- لوحة تحكم مطعم بسيطة
- لوحة إدارة بابلي بسيطة
- تقييمات ومراجعات أساسية

## لا تشمل:

- توصيل
- ERP كامل
- مخزون
- محاسبة متقدمة
- نظام موظفين متقدم
- فيديوهات قصيرة متقدمة
- AI معقد
